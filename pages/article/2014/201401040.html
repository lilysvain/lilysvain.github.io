CSS3-Speech Bubbles

以下实例出自《Stunning CSS3》一书的第2章，原文是英文版，我对它进行了简单的翻译，并将做出的实例用demo展示出来。

1. Progressive Enhancement（渐进增强）

Progressive enhancement 是一种网页开发方式。首先让网页在低版本浏览器和设备上显示且运行正常，然后逐层使用非必需、更高级的 CSS 和 JavaScript 功能为当前和未来的浏览器提供更好的支持。例如，你使用普通的语义化 HTML 构建一个表单，使它在没有使用 CSS 的情况下依然显示良好，然后使用 CSS 丰富它的外观，用 JS 提供客户端的验证以此提高可用性。该实例便是用 Progressive enhancement 方法开发。

2. Design in the Browser（在浏览器中进行设计）

Instead of using Photoshop or Fireworks to mock up your design as a static image, go straight to HTML and CSS to create the design mockup in its real, final medium.

直接使用 HTML 和 CSS 进行设计。在设计初期，你还是可以使用图形软件创建你的大致 idea；在 CSS 无法处理的时候，也要用图形软件绘制精细图形。

设计出来的图像通常会跟最后写出来的页面有出入，特别是如果你不想用过多图片。一些hover的动态效果也不容易在静态图片中展现出来。在一个人进行 design 和 coding 的时候，在 coding 的同时进行设计也会节省很多时间。

3. The Base Page（基本页面）

见 dome1，该页面在 IE8 以下的浏览器和小屏幕上的显示依然正常。

Demo1

3.1 word-wrap

该属性有两个值：默认值 normal 表示允许内容顶开或溢出容器边界；break-word 表示内容将在边界内换行，如果需要，单词内部允许断行。

通常需要用到 word-wrap 的场合为文本里含有 URL 时。如果 URL 里有 (-), 所有主流浏览器会在此处换行；Webkit 内核的浏览器和 IE 浏览器不会在 (/) 处换行；没有浏览器会在 (_) 处换行。

blockquote { ...... word-wrap: break-word; }

3.2 Creating Triangles out of borders（边框外小三角）

blockquote:after { content: "\00a0"; display: block; width: 0; height: 0; border-width: 10px 20px 10px 0; border-style: solid; border-color: transparent #A6DADC transparent transparent; position: absolute; top: 20px; left: -20px; }

3.3 CSS3's RGBA and HSLA syntax（两种支持透明度的颜色表示法）

RGBA 表示 red-green-blue-alpha，HSLA 表示 hue-saturation-lightness-alpha（色相-饱和度-亮度-透明度）。

background-color: hsla(182,44%,76%,.5); border-color: transparent hsla(182,44%,76%,.5) transparent transparent;

IE8 及以下不支持 RGBA 和 HSLA，为了兼容 IE：

blockquote { ...... background: rgb(166, 218, 220); background: hsla(182,44%,76%,.5); }

注意：使用hsl是无效的，会使背景色为透明，且第二个background不能写成background-color，IE6 和 IE7 会尝试应用hsla/rgba颜色，发现无法使用后最终不会表现出任何颜色。

4. 添加效果

4.1 Linear Gradients（线性渐变）

gradient 功能是 Image Values 模块的一部分，目前支持度还不高。W3C链接

blockquote { ...... background-image: -moz-linear-gradient(hsla(0,0%,100%,.6), hsla(0,0%,100%,0) 30px); background-image: -webkit-gradient(linear, 0 0, 0 30, from(hsla(0,0%,100%,.6)), to(hsla(0,0%,100%,0))); background-image: linear-gradient(hsla(0,0%,100%,.6), hsla(0,0%,100%,0) 30px); }

起始颜色是60%的白色，终点颜色为完全透明，且在距离 box 顶部30px的地方，终点颜色会充满余下的垂直空间。

4.2 The box-shadow Property（阴影）

blockquote:{ ...... -moz-box-shadow: 1px 1px 2px hsla(0,0%,0%,.3); -webkit-box-shadow: 1px 1px 2px hsla(0,0%,0%,.3); box-shadow: 1px 1px 2px hsla(0,0%,0%,.3); } blockquote:hover { top: -2px; left: -2px; -moz-box-shadow: 3px 3px 2px hsla(0,0%,0%,.3); -webkit-box-shadow: 3px 3px 2px hsla(0,0%,0%,.3); box-shadow: 3px 3px 2px hsla(0,0%,0%,.3); }

第一个数字表示水平偏移, 浏览器使阴影向右移距 box 右边缘1px；第二个数字表示竖直偏移量；第三个数字表示 blur 半径, 表示阴影延伸2px，数值越大阴影越大越柔和，数值为 0 造成锋利边缘。

可以使用 text-shadow 给文字也添加阴影：

text-shadow: 1px 1px 0 hsla(0,0%,100%,.7);

4.3 Transforming the Avatars（旋转头像）

.comment-meta img { -moz-transform: rotate(-5deg); -o-transform: rotate(-5deg); -webkit-transform: rotate(-5deg); transform: rotate(-5deg); }
