Java Graphics Programming

Java 1.0 就推出了用于基本 GUI 编程的类库，Sun 称之为 Abstract Window Toolkit (AWT)。后来，为了使界面组件能在不同的平台中有一样的外观和表现，Sun 和 NetScape 合作创造了一个用户接口库 "Swing"，Swing 在Java SE 1.2 中成为标准库的一部分。

Note：Swing并不能完全取代 AWT，它构建于 AWT构架之上。写一个 Swing 程序时，你还是在使用 AWT 的基础功能，尤其是事件处理。

1. Frame

在 Java 中，顶层窗口被称为 frame。API: javax.swing.JFrame

继承自：java.awt.Frame -》java.awt.Window -》java.awt.Container -》java.awt.Component

All Methods：

  • public void setVisible(boolean b) --- 继承自java.awt.Window。Frames start their life invisible. That gives the programmer the chance to add components into the frame before showing it for the first time.
