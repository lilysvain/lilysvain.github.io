Js中的CSS操作

我发现自己的前几篇文章整理的好乱啊，主要是太贪心，什么都想有，又什么都懒得讲清楚，我还是一点一点来好了。

为了尽量使行为和表现分离，JavaScript 中是不应该出现太多样式操作的。但是碰上想用 CSS3 写幻灯一类的动画时，不可避免需要 Js 控制一些参数，也就不可避免地需要用 Js 去操作 CSS，所以这块还是该理理。

HTML 元素在 JavaScript 中都有一个对应的 style 属性，这个 style 对象是 CSSStyleDeclaration 的实例，但只包含着通过 HTML 的 style 特性指定的样式信息，如下所示：

[div id="myDiv" style="background-color:blue;width:10px"][/div] alert(document.getElementById("myDIv").style.backgroundColor); //blue

通常 style 对象用于改变元素样式。改变后的样式信息也会及时反应到 HTML 元素上。

1. 计算的样式

因为 style 不包含那些从其他样式表层叠而来并影响到当前元素的样式信息，“DOM2 级样式”增强了 document.defaultView，提供了 getComputedStyle() 方法。这个方法接受两个参数：要取得计算样式的元素和一个伪元素字符串（例如“:after”），如果不需要伪元素信息，第二个参数可以是 null。getComputedStyle() 方法同样地返回的是一个 CSSStyleDeclaration 对象，其中包含当前元素的所有计算的样式。例如：

var st = document.defaultView.getComputedStyle(element,null); var matrix = st.getPropertyValue("-webkit-transform") || st.getPropertyValue("-moz-transform") || st.getPropertyValue("-ms-transform") || st.getPropertyValue("-o-transform") || st.getPropertyValue("transform") || "fails...";

matrix 将是一个字符串，如： "matrix(1,0,0,1,0,0)"
