正则表达式

在使用正则表达式（the regular expression）的各种程序和类库之间，表达式语法并未完全标准化。尽管在基本结构上达成了一致，但它们在细节上仍旧存在着许多令人抓狂的差异。该文中展示的是 java 语法中的正则表达式结构。

java.util.regex.Pattern

A regular expression, specified as a string, must first be compiled into an instance of this class. The resulting pattern can then be used to create a Matcher object that can match arbitrary character sequences against the regular expression.

A typical invocation（调用） sequence is thus：

Pattern p = Pattern.compile("a*b"); Matcher m = p.matcher("aaaaab"); boolean b = m.matches();

Note: 正则表达式字符串中如有“\”，须经过转义“\\”才能在compile方法中使用。

Pattern MethodsAPI
  • public static Pattern compile(String regex) --- Compiles the given regular expression into a pattern.
  • public String[] split(CharSequence input) --- Splits the given input sequence around matches of this pattern.

Summary of regular-expression constructs

java.lang.Character classes

  • \p{javaWhitespace} --- Equivalent to java.lang.Character.isWhitespace()

Greedy quantifiers

  • X+ --- X, one or more times

Boundary matchers

  • \A --- The beginning of the input

空白符的问题

（public static boolean isWhitespace(char ch)）A character is a Java whitespace character if and only if it satisfies one of the following criteria:

  • It is a Unicode space character (SPACE_SEPARATOR, LINE_SEPARATOR, or PARAGRAPH_SEPARATOR) but is not also a non-breaking space ('\u00A0', '\u2007', '\u202F').
  • It is '\t', U+0009 HORIZONTAL TABULATION.
  • It is '\n', U+000A LINE FEED.（换行）
  • It is '\u000B', U+000B VERTICAL TABULATION.（换行）
  • It is '\f', U+000C FORM FEED.（换页符、换行）
  • It is '\r', U+000D CARRIAGE RETURN.（回车符，换行）
  • It is '\u001C', U+001C FILE SEPARATOR.
  • It is '\u001D', U+001D GROUP SEPARATOR.
  • It is '\u001E', U+001E RECORD SEPARATOR.
  • It is '\u001F', U+001F UNIT SEPARATOR.

换行（newline，Line break，end-of-line（EOL））：换行字符可以看作是行的结束符，也可以看作行之间的分隔符，这两种处理方式之间存在一些歧义。如果换行字符被当作分隔符，那么文件的最后一行就不需要再有换行字符。但是多数系统的做法是在最后一行的后面也加上一个换行字符，也就是把换行字符看作是行的结束符。这样的程序在处理末行没有换行字符的文件时，可能会存在问题。相反地，有的程序把换行符看作分隔符，就会把最末尾的换行字符看作是新行的开始，也就是多出了一个空行。

通常利用转义序列\n来换行。而回车键（or 确认键：carriage return）用以把一设备的位置重设到一行字的头。用\r来表示回车，依照操作系统的不同，有时会配合换行字符\n使用。
