Java 线程

笔试面试中经常被问到多线程的问题，只看过相关章节和文章，自己思路还比较杂乱吧，一边整理一边更文，可能会比较慢。

线程的概念以及和进程的区别应该不用多说，参见操作系统。Java 有提供对多线程编程的支持，那么问题的开始，如何创建一个线程：

1. 创建线程的几种方式

1.1 实现 Runnable 接口

Runnable 接口很简单，其中只有一个 run() 方法。我们实现一个类，并实例化一个对象：

class MyRunnable implements Runnable { public void run() { task code } } Runnable r = new MyRunnable();

接下来创建一个 Thread 对象，并启动线程：

Thread t = new Thread(r); t.start();

1.2 继承Thread类

也可以直接实现一个 Thread 的子类：

class MyThread extends Thread { public void run() { task code } }

然后就可以创建该子类的对象并使用 start() 方法。
